### Server and Client &mdash;

ကျတော်တို့ လိုချင်တဲ့ content တွေ ဥပမာ photo, video and document တွေကို စုစည်းထားတဲ့ computer ကို server လိုခေါ်ပါတယ်။ (website တစ်ခုကို အသင့်ရေးဆွဲထားပီး သိမ်းဆည်းထားပါတယ်။ webpage ကို browser ကသာ နားလည်တဲ့ HTML, CSS and JS တို့ဖြင့် တည်ဆောက်ထားပါတယ်။)

ကျတော်တို့က request တစ်ခုကို ပို့လွှတ်လိုက်ပီးဆိုရင် server ပါ ကျတော် လိုချင်တဲ့ data တွေကို စုစည်းပီး ကျတော်တို့ရဲ့ browser ကို ပြန်ပို့ပေးလိုက်ပါတယ်။

### Static and Dynamic Website

ဒါကို static website လို့ခေါ်ပါတယ်။

Dynamic website ဆိုတာကတော့ content တွေကို လိုအပ်သလို ပြောင်းလဲပီး ပို့ပေးရပါတယ်။ ဥပမာ လူနှစ်ယောက်ရဲ facebook page လိုမျိုး တစ်ယောက်နဲ့ တစ်ယောက်ဆီမှာ ဖော်ပြပေးတဲ့ content တွေဟာ တူမှာမဟုတ်ပါဘူး။ Dynamic website တွေမှာ database နဲ့ backend language (Nodejs, PHP, ASP.net, Java) တစ်ခုခု အသုံးပြုထားပါတယ်။

### HTML - Basic Structure Language

HTML ဟာ web technology မှာ အခြေခံအကျဆုံး language တစ်ခုဖြစ်ပါတယ်။ သူနဲ့ content တွေကို ဖန်းတီးရတာဖြစ်ပါတယ်။

### CSS - Cascading Style Sheet

CSS ကတော့ HTML ဖြင့် ဖန်တီးထားသော content တွေကို လှပအောင် ပြုလုပ်ပေးပါတယ်။

### JS - JavaScript

သူကတော့ programming language တစ်ခုဖြစ်ပီး webpage ပေါ်က data တွေ element တွေကို manipulate လုပ်နိုင်ပေးပါတယ်။
