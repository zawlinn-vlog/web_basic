Semantic HTML ဆိုတာ ဘာလဲ?

  1. Semantic HTML ဆိုတာ HTML Element တစ်ခုချင်းစီဟာ သီးခြား အဓိပ္ပါယ်နဲ့ ရည်ရွယ်ချက် ရှိတယ်လို့ ဆိုလိုတာပါ။
  2. HTML Element တွေဟာ page ပေါ်မှာ ဘယ်လို Render လုပ်မလဲ ဆိုတာကိုပဲ မစဉ်းစားဘဲ Element က ဘာကို ဆိုလိုတာလဲ ဘာကို ကိုယ်စားပြုတာလည်းဆိုတာကိုပါ စဉ်းစားသင့်ပါတယ်။
