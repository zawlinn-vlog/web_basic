## Setting Up Our Code Editor &mdash;

1. Download and Install Vs Code.
2. go to extension and Install `Prettier`
3. go to setting. type in search bar
   1. Default format -> choose Prettier - Code Formatter
   2. format on save -> Auto Save -> choose onFocusChange
   3. tab size -> 2
4. Install one `monokai theme` from extension
5. Install `vscode-icons` and set icon
6. Install `image preview`
7. Install `Color Hightlight`
8. Install `Auto Rename Tag`

### Create Project Folder &mdash;

1. Create a Folder name web_basic
2. Vs Code -> File -> Open Folder -> Choose the folder that we create
