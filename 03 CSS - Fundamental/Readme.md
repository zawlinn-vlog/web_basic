### What is CSS?

- CSS Stand for Cascading Style Sheets.
- CSS describes the visual style and presentation of the content in written HTML
- CSS consists of countless properties that the developer use to format the content: properties about font, text, layout and etc.

```css
h1 {
  color: blue;
  font-size: 20px;
  text-align: center;
}

/* 
    h1 is selector 
    color: blue; decoration
    decoration has property: value;
*/
```
